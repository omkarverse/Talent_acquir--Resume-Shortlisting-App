## ðŸ“š About the Project

Hiring teams receive tons of resumes, and manually checking each one is slow, repetitive, and prone to mistakes. Talent Acquire solves this by:

* Presenting every candidate as a visual profile card
* Allowing recruiters to quickly swipe right to shortlist or swipe left to reject
* Supporting **AI-style ranking** for better decisions
* Delivering a polished, swipe-friendly mobile experience

This app demonstrates how modern UI/UX patterns can make recruitment tools more efficient and enjoyable.
